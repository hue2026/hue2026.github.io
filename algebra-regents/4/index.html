🔑 Key Vocabulary
System of Equations
Two or more equations with the same variables. A solution must satisfy ALL equations simultaneously.
Solution of a System
The ordered pair (x, y) that makes every equation true at the same time — the intersection point.
Consistent vs. Inconsistent
Consistent: has at least one solution (lines cross or overlap). Inconsistent: no solution (parallel lines).
{ y = 2x + 1  |  y = −x + 7 }
System of 2 linear equations → find (x, y) that works for both
Part 1 · Systems Defined

What Is a System of Equations?

Look at this graph. Both lines pass through the point (3, 4). What's special about that point? Why can't we just call any other point on Line 1 a "solution to the system"?
Think of a system as a competition between two rules. Each equation is a constraint. The solution is the one (x, y) that obeys both rules at the same time.

Like two GPS paths meeting at the same street corner — that intersection is the answer!
Solution = Intersection Point
⚠️ Regents Trap
A solution must satisfy both equations. Always check by substituting back into BOTH equations, not just one!

Example System
y = 2x − 1
y = −x + 5

Step 2: Plot y-intercept (0, b)
Step 3: Use slope rise/run
Step 4: Find intersection!
⚠️ Graphing Limitation
Graphing is great for visual insight but can be imprecise for non-integer solutions. Use algebra methods when you need exact values!
Part 2 · System Types

Three Personalities of Systems 🎭

I'll describe a real-world scenario — tell me which type of system it represents: "Two cell phone plans that always cost the same amount no matter how many minutes you use." What about two plans that can never cost the same?
Quick Classifier — Compare slopes and y-intercepts:
  • Different slopes → 1 solution (intersecting)
  • Same slope, diff. y-int → 0 solutions (parallel)
  • Same slope, same y-int → ∞ solutions (same line)
⚠️ Regents Trap
An inconsistent system does NOT have "no solution to either equation" — each line is a valid equation. It's the system with no solution because they never share a point!

Part 3 · Method 2

Substitution Method 🔄

Imagine you know that x = y + 3. If you see "x" somewhere else, what would happen if you replaced it with "y + 3"? Why would that be helpful?
Example System
Eq. 1: x = 2y − 1
Eq. 2: 3x + y = 12
Think of substitution as a sneaky swap: if you know what x equals (in terms of y), you're allowed to secretly swap x with that expression anywhere else in the problem. Suddenly you have just one variable — and that's easy!
Step 1: Isolate one variable
Step 2: Substitute into OTHER equation
Step 3: Solve for remaining variable
Step 4: Back-substitute to find the other
Step 5: CHECK in both original equations!
Step-by-Step Solution
x = 2y − 1  (already isolated!)
Sub into Eq. 2: 3(2y−1) + y = 12
Distribute: 6y − 3 + y = 12
Combine: 7y − 3 = 12
7y = 15 → y = 15/7... wait—
Let's use a cleaner example:
Cleaner Example:
y = 3x − 1   and   2x + y = 9
Sub: 2x + (3x−1) = 9
5x − 1 = 9 → 5x = 10 → x = 2
y = 3(2) − 1 = y = 5
Check: 2(2)+5=9 ✓   5=3(2)−1 ✓
⚠️ Common Mistake
After solving for x, substitute back into the original equation — not your rearranged one. And always check both!

Part 3 · Method 3

Elimination Method ❌

If I add 5 apples to a bag, and then remove 5 apples from the same bag, what do I get? How does that idea apply to equations?
Example System
Eq. 1: 3x + 2y = 12
Eq. 2: 3x − 2y = 0
Think of elimination as a magic eraser: if you add two equations and one variable completely cancels out (0y!), you're left with just x. Two unknowns become one in a single move!
Step 1: Line up equations vertically
Step 2: Make one variable's coefficients cancel
Step 3: ADD the equations (variable eliminated!)
Step 4: Solve the one-variable equation
Step 5: Back-sub and CHECK both!
Adding the equations:
  3x + 2y = 12
+ 3x − 2y =  0
= 6x + 0y = 12
→ x = 2
Sub back: 3(2) + 2y = 12 → 2y = 6 → y = 3
Answer: (2, 3) ✓ Check: 6+6=12 ✓ 6−6=0 ✓
When coefficients DON'T cancel:
2x + 3y = 11   and   4x − y = 5
Multiply Eq.2 by 3: 12x − 3y = 15
Now add: 14x = 26 → x = 13/7...
⚠️ Multiplication Trap
When multiplying an equation to match coefficients, multiply EVERY term (both sides). Forgetting the constant is the #1 Regents error on elimination!

Which Method Should You Choose? 🤔

📐
Graphing

✅ You want a visual picture

🔄
Substitution
✅ One variable is already isolated
✅ One variable has coefficient of 1
✅ Word problems (define one var)
❌ Both equations have big coefficients
x = y − 3

✅ Coefficients can cancel easily
✅ Both equations in standard form
✅ Faster when set up nicely

Part 4 · Real World

📱 Cell Phone Plans — NYC Real Life

The Scenario 📱
Plan A: $20/month + $0.10 per text
Plan B: $35/month + $0 per text (unlimited)
Before writing any equation: at what number of texts do you think both plans cost the same? Is there a number of texts where Plan A costs more? What does that "break-even" point mean in this context?
Build the System
Let x = number of texts per month
Let y = total monthly cost
Plan A: y = 0.10x + 20
Plan B: y = 35
0.10x + 20 = 35 → x = 150 texts
At 150 texts/month, both plans cost $35
Now you decide: If you send 200 texts/month, which plan is cheaper? If you send 50 texts, which plan wins?
