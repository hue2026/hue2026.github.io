Adding Polynomials

The Golden Rule: Like Terms Only!

Step-by-Step Example
Add: (3x² + 2x − 1) + (x² − 5x + 4)
1 Drop the parentheses (+ sign = no change)
3x² + 2x − 1 + x² − 5x + 4
2 Group like terms by color/degree
3x²+x² + 2x−5x + −1+4
3 Combine like terms
= 4x² − 3x + 3 ✓
What Are "Like Terms"?
Think of them like fruit:
🍎 x² terms add with x² terms
🍊 x terms add with x terms
🍋 Constants add with constants
❌ x² + x ≠ 2x³ (different degrees!)
Regents Tip:
Always write your answer in standard form — highest degree first!
What happens if none of the terms are alike? Can you still add the polynomials?

Subtracting Polynomials

The Sign Trap — Most Common Regents Error!

Subtract: (5x² + 3x) − (2x² − x + 4)
❌ Common Mistake:
5x² + 3x − 2x² − x + 4   (only changed first sign!)
= 3x² + 2x + 4  ✗ WRONG
✅ Correct Method:
Step 1: Distribute the negative to ALL terms
5x² + 3x − 2x² + x − 4
Step 2: Combine like terms
= 3x² + 4x − 4 ✓
The "Flip Every Sign" Trick
When you see −(ax² + bx + c), it becomes:
−ax² − bx − c
Every single sign flips. Think of −1 multiplying the whole group.
Memory Hook:
"Minus a group = flip the whole group." Say it out loud every time!
Why does subtracting a negative become addition? Where does this rule come from?

Multiplying Polynomials · Part 1

Monomial × Polynomial: Distribute!

The Distributive Property
3x(x² − 2x + 3)
= 3x·x² + 3x·(−2x) + 3x·3
= 3x³ − 6x² + 9x
= 3x³ − 6x² + 9x ✓
Rules for Multiplying Powers
xᵃ · xᵇ = xᵃ⁺ᵇ
Add exponents when multiplying same base
Example: x² · x³ = x⁵ (not x⁶!)
Add, don't multiply the exponents
Also multiply the coefficients:
3x · 4x² = 12x³
(3×4=12, x¹⁺²=x³)
Why do we add exponents when multiplying? What does x² × x³ really mean in expanded form?
Multiplying Polynomials · Part 2

FOIL: Binomial × Binomial

Multiply: (x + 3)(x + 2)
F: x · x = x²
O: x · 2 = 2x
I: 3 · x = 3x
L: 3 · 2 = 6
x² + 2x + 3x + 6 = x² + 5x + 6 ✓
FOIL = Organized Distributive Property
Why FOIL works:
Each term in the first binomial must multiply every term in the second. FOIL makes sure you never miss one.
Don't forget to combine like terms!
2x + 3x = 5x (the middle terms always combine)
Now try: (x − 4)(x + 1)
Watch the negative sign on −4!
Is FOIL only for binomials? What would you do with (x + 1)(x² + x + 2)?

Special Products

Patterns Worth Memorizing

🟦
Perfect Square
(a + b)² = a²+2ab+b²
Example:
(x+3)²=x²+6x+9
The middle term = 2·a·b
⬛
Difference of Squares
(a+b)(a−b)=a²−b²
Example:
(x+5)(x−5)=x²−25
Middle terms cancel out!
💡
Why It Matters
These patterns appear on every Regents exam. Recognizing them saves time.
Also helps with factoring — our next unit!
What happens to (a − b)²? Is it a²− b²?

Real-World Application

The Garden Plot Problem 🌱

Problem:
A rectangular garden has a length of (x + 5) feet and a width of (x + 3) feet. Write a polynomial for the area. Then find the area when x = 4.
Solution Steps:
Area = Length × Width
= (x + 5)(x + 3)
Apply FOIL:
x² + 3x + 5x + 15 = x² + 8x + 15
When x = 4:
4² + 8(4) + 15 = 16 + 32 + 15 = 63 sq ft
Extension:
If the garden needs a path 1 ft wide around it, write a polynomial for the total area including the path.
Does the area formula x² + 8x + 15 still work for x = 10? x = 0? What does x = 0 mean in context?

Watch Out!

Top 5 Polynomial Traps on the Regents

⚠️
Trap 1: Forgetting to flip all signs when subtracting
❌ (x²+3x) − (2x−1) = x²+3x−2x−1   ✅ = x²+3x−2x+1
⚠️
Trap 2: Adding unlike terms
❌ 3x² + 2x = 5x³   ✅ They can't be combined — different degrees!
⚠️
Trap 3: Multiplying exponents instead of adding
❌ x² · x³ = x⁶   ✅ x² · x³ = x⁵ (add exponents!)
⚠️
Trap 4: Missing the FOIL "Outer" or "Inner" step
❌ (x+3)(x+2) = x²+6   ✅ = x²+5x+6 (don't skip the middle!)
⚠️
Trap 5: Not writing in standard form
❌ 3 − 2x + x²   ✅ x² − 2x + 3 (highest degree first!)
Which of these traps have you already made today? How will you catch yourself next time?
