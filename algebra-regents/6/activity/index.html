Method Find shared number Find shared variable(s) GCF = both combined Check: distribute back ✓
How To Factor GCF
Step 1 — What number divides into ALL coefficients?
Step 2 — What's the lowest power of each shared variable?
Step 3 — Multiply those together = your GCF.
Step 4 — Divide every term by the GCF; write what's left in ( ).
⚠ Trap: Don't grab only the number or only the variable — take both. In 6x² + 9x the GCF is 3x, not just 3 and not just x.

Part B — Trinomial Factoring: The Two-Number Detective

6 Problems · ~14 min

Find the detective pair — two numbers that MULTIPLY to c and ADD to b — then write the factored form.

Formula x² + bx + c = (x + m)(x + n) m · n = c m + n = b c > 0: signs match b  |  c < 0: signs differ
Sign Detective Rules
c is POSITIVE → both signs match b
x² + 7x + 10 → both +, since b > 0
x² − 7x + 10 → both −, since b < 0
c is NEGATIVE → signs differ
Bigger number takes the sign of b
x² + 3x − 10 → (x + 5)(x − 2)
💭 List factor pairs of c first — then scan each pair's sum for b.

Part C — Solving with the Zero Product Property

5 Problems · ~12 min

Solve each quadratic equation. Always set equal to 0 FIRST, then factor, then split.

5-Step Rule ① Set = 0 ② GCF out ③ Factor trinomial ④ Each factor = 0 ⑤ Solve & CHECK
Why Zero Is Special
If A · B = 0, then A = 0 or B = 0
Zero is the only number you can't sneak up on by multiplying. If a product is zero, one factor MUST be zero — that certainty turns one hard equation into two easy ones.
⚠ #1 Trap: (x + 2)(x − 5) = 4 does not mean x + 2 = 4.
Zero Product Property only fires when the product equals ZERO. Move everything to one side first!
