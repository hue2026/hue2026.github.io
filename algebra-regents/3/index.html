Part 1 · Do Now

Activate Prior Knowledge

Do Now Problem
Solve for y:   2x + 3y = 6
Hint: isolate y — get it alone on one side!
✅ Answer
Subtract 2x → 3y = −2x + 6 → divide by 3 → y = −⅔x + 2. Now it's in slope-intercept form!
What is the slope in your answer? What is the y-intercept? Where would you start if you were going to draw this line on a graph?
🔑 Key Vocabulary
Linear Function
A relationship where the rate of change is constant. Its graph is always a straight line.
Slope (m)
How steep the line is. Calculated as rise ÷ run = change in y ÷ change in x.
y-intercept (b)
Where the line crosses the y-axis. The starting value when x = 0.
y = mx + b
m = slope  |  b = y-intercept
Part 2 · Slope

What is Slope? Rise Over Run 🏔️

Look at this graph. If you were walking along this line from left to right, would you be going uphill or downhill? How steep is it compared to a staircase?
Slope tells you the rate of change: for every 1 unit you move RIGHT along the x-axis, how many units do you move UP (or down)?

Think of slope as the personality of a line — it tells you everything about how the line behaves!
m = rise/run = (y₂ − y₁) / (x₂ − x₁)
📏 Memory Trick
Rise = vertical (up/down)  |  Run = horizontal (left/right). Rise comes before Run — y comes before x in the formula!
Part 2 · Slope Types

The Four Slope Personalities

I'm going to describe a real-world scenario — tell me which type of slope it has: "A ski slope going straight down a mountain face". What about a flat road? A vertical cliff?
  • Positive: going uphill 🏔️ — price increasing, temperature rising
  • Negative: going downhill ⛷️ — cost decreasing, time running out
  • Zero: flat road 🛣️ — no change, constant value
  • Undefined: vertical wall 🧱 — x never changes, equation is x = c
⚠️ Regents Trap
"Zero slope" ≠ "no slope"! Zero slope means the line IS horizontal. "No slope" or "undefined slope" means the line is VERTICAL. They're completely different!

Part 3 · Slope-Intercept Form

Decoding y = mx + b 🔓

Look at the equation y = 2x + 1. Before graphing, what can you predict about the line? Will it go uphill or downhill? Where will it cross the y-axis?
Think of y = mx + b like a recipe:
• b = where you START (y-intercept) 📍
• m = how you MOVE from there (slope) 🚶

Start at (0, b), then use the slope fraction to find your next point: go UP by the numerator, RIGHT by the denominator.
Start at b → move by m
📌 Graphing Steps
1. Plot the y-intercept (0, b)   2. Use slope (rise/run) to find 2nd point   3. Draw a straight line through both points with arrows!

Part 4 · Real World

🚕 Taxi Ride: Linear Functions in Real Life

🗺️ The Scenario
A NYC taxi charges a $3.00 base fare plus $2.50 per mile.
Before writing any equation — what does the $3.00 represent on a graph? What does the $2.50 represent? Which one is the slope and which is the y-intercept?
Build the Equation:
• y = total cost ($)   x = miles traveled
• slope (m) = $2.50/mile (rate of change)
• y-intercept (b) = $3.00 (base fare, when x=0)
y = 2.50x + 3.00

As you increase m from 1 to 5, what changes? As you change b from 0 to 3, what changes? What stays the same when you only change b?
Prediction questions:
  • What happens when m = 0?
  • Can m be a decimal like 0.5?
  • What do two parallel lines have in common?
💡 Key Insight
Parallel lines have the same slope but different y-intercepts. Perpendicular lines have slopes that multiply to −1 (negative reciprocals).
Part 4 · Real World Video

🎬 Word Problems: Slope-Intercept in Context
⏱ 4 min · Slope-Intercept Form Word Problems (YouTube)
Key takeaway from the video: The keyword "per" signals the slope. Words like "initial," "flat fee," or "startup" signal the y-intercept.
🔑 Slope ↔ Rate Translator
Slope = rate of change:
• "$2.50 per mile" → m = 2.50
• "$40 per month" → m = 40
• "5 inches per year" → m = 5

Y-intercept = starting value:
• "$3 base fare" → b = 3
• "10-gallon tank to start" → b = 10
If you were told "a candle is 12 inches tall and burns 1.5 inches per hour," what would be the equation? What does x represent? What does y = 0 mean in this context?
