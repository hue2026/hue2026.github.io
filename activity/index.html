🎯 Grades 3–5 🐱 scratch.mit.edu ⏱️ ~45–60 min

🔑 Part 1 · Key Vocabulary

Match or fill in each definition in your own words below the card.

🎮 Game Engine
Software that handles graphics, physics, sound & movement so developers can build games faster.
🐱 Scratch
A drag-and-drop block coding tool made by MIT — no typing needed!
🧩 Sprite
Any object in a game: a character, enemy, coin, or power-up.
🔄 Iterative Design
Make it → test it → get feedback → improve it. Repeat!
⚡ Unity
A professional game engine used for big 2D/3D games like Among Us and Cuphead.
🕹️ Godot
A free, open-source engine — great for small teams and learning game logic.
