🧩 Sprite

🤖 What is a game engine?

It's like the engine in a car 🚗

A game engine is the hidden brain that makes games run. It handles:

  • Moving characters
  • Detecting collisions
  • Playing sounds & music
  • Displaying art

Without engine → you'd have to code everything from scratch!

🐱 Meet Scratch

Drag-and-drop coding blocks. Made by MIT. Ideal for ages 8–16. Create stories, animations, games without typing code!

🖥️ Scratch Editor has 3 parts:

🧩 Block Palette — drag code blocks
🐱 Sprite Area — your characters
🎬 Stage — where the game plays

Try it free at scratch.mit.edu →

⚡ Unity · Real power

Used by professionals! Games like Among Us, Cuphead, Hollow Knight. Supports 2D, 3D, VR. You write C# code.

⚡ Built with Unity:

🎮 Among Us · Cuphead · Hollow Knight

Unity isn't just for games!

🕹️ Godot · Free & open source

Totally free! Lightweight, great for 2D games. Uses its own scripting language (GDscript) similar to Python. Growing community.

🎨 Games like "Celeste" (prototype), "Brotato" made with Godot! Perfect for learning game dev without costs.

⚖️ Compare: Scratch vs Unity vs Godot
🐱 Scratch

✅ No install, block coding, instant sharing. Great for beginners & school projects.

⚡ Unity

✅ Professional 2D/3D, huge asset store. Needs coding (C#). Ideal bigger games.

🕹️ Godot

✅ Fully free, lightweight, friendly scripting. Perfect for 2D & learning real engine logic.

💻 Let's build! Simple Scratch Game

Step 1

Open scratch.mit.edu → Create → Choose a sprite (cat or any character).

Step 2

Add “when green flag clicked” + “forever” + “if key (right arrow) pressed, change x by 10”.

Step 3

Add another sprite as a “treasure”. Add point system: “if touching treasure, change score by 1”.

Step 4

Test! Does it work? Iterate: adjust speed, add sounds, or walls.

🔄 Iterative Design · test · improve
🔁

Prototype → Playtest → Feedback → Refine

Game designers never finish at the first try! The best games are improved again and again.
